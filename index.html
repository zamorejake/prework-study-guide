Prework Study Guide

HTML

  • It is important to delcare the document type at the top of the HTML file
  • Different sizes of lettering can be changed with H1-H6 ideally in desending order
  • The head tag is not shown to the end user
  • It is highly recommended to add an alt tag to any image for screen readers or anyone that cannot load an image due to internet

CSS

  • When declaring colors either the name of the color can be used or the HEX code for it
  • In the CSS file when refering to the class a . is used
  • Margin works very similarly to a word document's margin, while padding is used inside the element to add spacing
  • Make sure not to use DRY code, in this style sheet we combined header and footer

Git

  • It is very important to not push to main!
  • When commiting to a branch it is recommended to add a note along with it
  • When starting back up on a project it is best practice to pull from the active branch to make sure you are not working on old code
